# فلو صفحة الضيف — مساران ونهاية مشتركة

## المسار ١: عضو جديد (تسجيل مجاناً)
- **البداية:** زر «اضغط للتسجيل مجاناً»
- **الخطوة:** نموذج واحد: اسم + رقم الجوال + هوية (بدون ادخل رقمك فقط)
- **عند الإرسال:** حفظ عضوية فضية + تصدير جوجل شيت + فتح واتساب للاستقبال (كود: —، جائزة: عضوية إليت الفضية)
- **النهاية:** شاشة نجاح «تم وصول الرسالة للاستقبال»

## المسار ٢: نزيل قديم أو جديد (اضغط هنا)
- **البداية:** زر «اضغط هنا»
- **الخطوة ١:** ادخل رقمك فقط (للتحقق من قاعدة البيانات)
- **عند الإرسال:**
  - **لو وُجد (عضو):** العجلة تدور تلقائياً → كود + جائزة → ثم شاشة «فئتك + إرسال واتساب» (بدون طلب الرقم مرة ثانية)
  - **لو لم يُوجد (جديد):** نموذج تسجيل (اسم + رقم + هوية) → إرسال → حفظ فضية + تصدير شيت → العجلة تفتح ويدور → كود + جائزة → ثم شاشة «فئتك + إرسال واتساب»
- **النهاية:** شاشة «فئتك» + زر إرسال واتساب للاستقبال، ثم شاشة نجاح

## النهاية المشتركة للمسارين
- كود (حقيقي من العجلة أو «—» للتسجيل المجاني)
- رسالة واتساب للاستقبال (الاسم، الجوال، الفئة، الجائزة، الكود)
- تحديث جوجل شيت (فقط عند تسجيل عضو جديد)
